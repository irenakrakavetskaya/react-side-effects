# React Side Effects (PlacePicker)

A small **React + Vite** project that demonstrates how to handle side effects in React: timers with cleanup, browser APIs (geolocation), localStorage persistence, and portals / native dialogs.

**HTTP / Backend:** This project does **not** rely on a backend server or HTTP requests. Place data is loaded from a local dataset (`src/data.js`) and the app uses browser APIs like `navigator.geolocation` and `localStorage` on the client.

## Tech Stack

- React
- Vite

### Backend install

```bash
npm install
node app.js
```

### Frontend install

```bash
npm install
npm run dev
```
